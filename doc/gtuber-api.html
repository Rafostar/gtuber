API Reference

Gtuber — misc functions
GtuberClient — a web client that fetches media info
GtuberMediaInfo — contains media information
GtuberStream — represents a direct media stream
GtuberAdaptiveStream — represents an adaptive media stream, usually for use in an adaptive streaming manifest
GtuberManifestGenerator — generates adaptive streams manifest from media info

Those functions are publically available as part of Gtuber API in both header files and generated introspection bindings.
